2 T-Tests

2.1 Compare Magnitudes Between States

Are California and Nevada earthquakes different?

# Separate by state
ca_data <- earthquake %>% filter(State == "CA")
nv_data <- earthquake %>% filter(State == "NV")

# Perform t-test
t_test_result <- t.test(ca_data$Magnitude, nv_data$Magnitude)

cat("T-test Results:\n")

3 ANOVA Test

3.1 Compare Intensity Across Distance Categories

Does intensity differ by distance?

intensity_data <- earthquake %>% filter(!is.na(Intensity))

# Perform ANOVA

6 Confidence Intervals

6.1 95% Confidence Interval for Magnitude

# Calculate confidence interval
mean_mag <- mean(earthquake$Magnitude, na.rm = TRUE)
sd_mag <- sd(earthquake$Magnitude, na.rm = TRUE)
n <- sum(!is.na(earthquake$Magnitude))
se <- sd_mag / sqrt(n)
ci_lower <- mean_mag - 1.96 * se
ci_upper <- mean_mag + 1.96 * se

cat("Magnitude Confidence Interval (95%):\n")
Magnitude Confidence Interval (95%):
cat("=====================================\n")
=====================================
cat("Mean:", round(mean_mag, 3), "\n")
Mean: 7.41 
cat("95% CI: [", round(ci_lower, 3), ",", round(ci_upper, 3), "]\n")
95% CI: [ 7.403 , 7.417 ]

Interpretation: We’re 95% confident the true mean magnitude falls in this range.
