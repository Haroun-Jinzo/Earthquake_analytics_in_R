2 Correlation Matrix

2.1 Calculate Correlations

# Select numeric columns
numeric_data <- earthquake %>%
  select(Magnitude, Distance_km, Intensity, Epicenter_Lat, Epicenter_Lon) %>%
  na.omit()

# Calculate correlation matrix
cor_matrix <- cor(numeric_data)

# Show as table
cor_matrix %>%
  round(3) %>%
  kable()

2.2 Visualize Correlation Matrix

corrplot(cor_matrix,
         method = "circle",
         type = "upper",
         tl.col = "black",
         addCoef.col = "black",
         title = "Correlation Matrix")

How to read: - Red = positive correlation - Blue = negative correlation - Bigger circles = stronger relationship

3.1 Magnitude vs Distance

ggplot(earthquake, aes(x = Distance_km, y = Magnitude)) +
  geom_point(alpha = 0.5, color = "darkblue") +
  geom_smooth(method = "lm", color = "red") +
  labs(
    title = "Magnitude vs Distance from Epicenter",
    x = "Distance (km)",
    y = "Magnitude"

  theme_minimal()

5 Conclusions

The correlation analysis confirms expected earthquake behavior:

  1. Magnitude is consistent across all stations (same earthquake)
  2. Larger earthquakes produce higher intensities at stations
  3. Intensity decreases as you move away from the epicenter

All relationships are statistically significant (p < 0.05)!
